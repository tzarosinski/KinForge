ParableForge Content Manager

The local Pages CMS integration is currently experiencing issues. Instead, you can use the official Pages CMS web application to manage your content.

Open Pages CMS Web App

How to use the web version:

  1. Click the button above to open the Pages CMS web app
  2. Sign in with your GitHub account when prompted
  3. Add your repository (ParableForge) if it's not already connected
  4. The web app will automatically detect your .pages.yml configuration
  5. You can now edit your adventure content through the web interface

Note: Changes made through the web app will create commits directly to your GitHub repository. After making changes, you'll need to pull the updates to your local repository.
